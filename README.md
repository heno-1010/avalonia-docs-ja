# このサイトについて

このサイトはAvalonia UI の公式ドキュメントを日本語で読めるようにすることを目的とした、非公式の翻訳プロジェクトです。

元のドキュメントはこちら：
- https://docs.avaloniaui.net/

## ⚠️ 注意

本サイトは非公式です。  
内容は公式ドキュメントを元にしていますが、正確性や最新性は保証されません。

また、公式の変更により内容が古くなる可能性があります。

## 📚 使い方
左側のサイドバーから各トピックを選択してください。

## 🔗 プロジェクト

翻訳データやサイトのソースコードは GitHub で公開しています。  
→ [GitHub Repository](https://github.com/heno-1010/avalonia-docs-ja)  
誤訳の報告や改善提案も歓迎しています。

## 🌍 翻訳範囲について
現在は Avalonia UI 公式ドキュメントの主要セクションを順次翻訳しています。  
すべてのページが完全に翻訳されているわけではありません。

## 🔄 更新について
公式ドキュメントの更新にできる限り追従していますが、反映には時間差があります。
